# Demerge :camera:
app para fotos, usando node.js, express, pug e mongodb

## Funções do Demerge

- [x] login/cadastro
- [x] postar foto
  - [x] adicionar legenda
- [x] curtir foto
- [ ] comentar na foto
- [x] ver número de "seguidores" no perfil
- [x] ver número de "seguindo" no perfil
- [x] ver número de "fotos" no perfil
- [ ] mudar foto de perfil
- [x] histórico de notificação

## Instalação
```
npm i
```
```
node app
```
